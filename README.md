#### Description:
    Page-loader is a CLI utility to download web page into a specified directory
### Package installation:
```bash
pip3 install git+https://github.com/FeoktistovAE/python-project-51
```

```bash
make setup
```
### Utility usage:
```bash
page-loader -h
```

```bash
usage: page-loader [-h] [-o OUTPUT] url
```
